# isodrive (halium)

* `sudo apt install build-essential`

* `git clone https://github.com/nitanmarcel/isodrive`

* `cd isodrive`

* `make`

* `sudo make install` (optional)

mount

* `sudo isodrive /full/path/to/file.iso`

umount

* `sudo isodrive`
